# Curitiba Inox App

Sistema de gestão de ordens de serviço — Curitiba Inox.

## Stack

- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (self-hosted)

## Desenvolvimento

```sh
npm install
npm run dev
```

## Deploy

```sh
npm run build
# scp dist/ para VPS → /var/www/app
```
